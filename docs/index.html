Tableau de bord interactif des valeurs cotées à la Bourse Régionale des Valeurs Mobilières

Bienvenue sur le tableau de bord d'analyse des valeurs de la BRVM

Cette page est en cours de chargement ou en attente de la première mise à jour de données.

Pour générer le tableau de bord avec les données les plus récentes, exécutez le script suivant :

python scripts/update_dashboard.py

Ce script va :

  • Récupérer les données actuelles des valeurs cotées à la BRVM
  • Collecter les capitalisations boursières, PER et dividendes
  • Générer un tableau de bord interactif
  • Mettre à jour cette page avec les données récentes
Note: Pour voir une mise à jour manuelle des données, veuillez consulter la section "Actions" du dépôt GitHub.
